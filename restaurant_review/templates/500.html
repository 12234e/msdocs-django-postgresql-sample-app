Oops, there was a problem

Try the following troubleshooting tips if running locally:
  • Check that an .env file exists and defines the environment variables DBHOST, DBNAME, DBUSER, and DBPASS.
  • Check that you ran a migration first with "python manage.py migrate".
  • Check that the database exists in your local PostgresSQL server.
Try the following troubleshooting tips if running in Azure App Service:
  • Check that the environment variables (DBHOST, DBNAME, DBUSER and DBPASS) are defined as configuration app settings for the service.
  • Check that you ran a migration first with "python manage.py migrate".
  • Check that the database referenced in DBNAME exists in the PostgresSQL service.
